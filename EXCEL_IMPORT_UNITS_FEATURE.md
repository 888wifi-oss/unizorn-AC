# 🎯 ฟีเจอร์การนำเข้า Excel สำหรับหน้า Units

## ✅ สิ่งที่ทำแล้ว
1. **สร้าง utility สำหรับอ่าน Excel** - `lib/utils/excel-import.ts`
2. **เพิ่ม icons** - Upload และ FileSpreadsheet ในหน้า Units

## 📋 สิ่งที่ต้องทำต่อ
1. เพิ่ม Dialog สำหรับการนำเข้า Excel ในหน้า `app/(admin)/units/page.tsx`
2. เพิ่ม state สำหรับจัดการไฟล์ Excel
3. เพิ่มฟังก์ชันสำหรับ import ข้อมูล
4. เพิ่มปุ่ม "นำเข้าจาก Excel" ในหน้า Units

## 🎯 วิธีใช้งาน
1. คลิกปุ่ม "นำเข้าจาก Excel"
2. เลือกไฟล์ Excel ที่เตรียมไว้
3. ตรวจสอบข้อมูลก่อนนำเข้า
4. คลิก "นำเข้าข้อมูล"

## 📊 Template Excel ที่ต้องใช้
- หมายเลขห้อง (Unit Number)
- ชั้น (Floor)  
- ขนาดห้อง (Size)
- ชื่อเจ้าของ (Owner Name)
- อีเมล (Email)
- เบอร์โทรศัพท์ (Phone)
- สถานะ (Status)

## ✨ ผลลัพธ์
- สามารถนำเข้าข้อมูลห้องชุดหลายห้องพร้อมกัน
- ประหยัดเวลาในการสร้างข้อมูล
- ลดโอกาสผิดพลาดในการกรอกข้อมูล
