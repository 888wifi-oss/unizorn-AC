# 🎯 ฟีเจอร์ Export QR Code for Invitation Codes

## แนวทางที่แนะนำ

### ตัวเลือก 1: Export QR Code เป็น PDF (แนะนำ)
**ข้อดี**:
- ✅ พิมพ์ได้เลย ไม่ต้องปริ้นทีละใบ
- ✅ แจกจ่ายง่าย
- ✅ มีข้อความอธิบายได้
- ✅ เต็มพื้นที่ใช้สอย

**ฟีเจอร์ที่ควรมี**:
- เลือกรายการที่ต้องการ Export (เลือกทั้งหมด หรือเลือกเฉพาะบางรายการ)
- Export เป็น PDF พร้อม:
  - QR Code สำหรับแต่ละรายการ
  - ชื่อ-นามสกุล
  - หมายเลขห้อง
  - รหัสเชิญ (text)
  - URL สำหรับลงทะเบียน
  - ข้อมูลการติดต่อ (อีเมล, เบอร์โทร)

### ตัวเลือก 2: Export เป็น Image (QR Code อย่างเดียว)
**ข้อดี**:
- ✅ ไฟล์เล็ก
- ✅ แชร์ผ่าน LINE/Email ได้ง่าย
- ✅ ใช้ได้กับระบบอื่นได้

**ฟีเจอร์ที่ควรมี**:
- Export แต่ละรายการเป็นไฟล์ Image แยกกัน
- Zip ไฟล์ทั้งหมดเข้าด้วยกัน
- ไฟล์ .png หรือ .jpg

### ตัวเลือก 3: Export เป็น CSV + QR Codes
**ข้อดี**:
- ✅ มีข้อมูลครบถ้วนใน Spreadsheet
- ✅ QR Code แยกเป็นไฟล์ Image
- ✅ นำไปใช้ในระบบอื่นได้

**ฟีเจอร์ที่ควรมี**:
- Export CSV พร้อม columns:
  - หมายเลขห้อง
  - ชื่อเจ้าของ/ผู้เช่า
  - อีเมล
  - เบอร์โทร
  - รหัสเชิญ
  - URL
  - ชื่อไฟล์ QR Code
- Export QR Code Images แยกเป็นไฟล์ .png

## 🎯 แนะนำ: ใช้ตัวเลือก 1 (PDF) + ตัวเลือก 3 (CSV + QR Codes)

### ทำไม?
- PDF เหมาะสำหรับพิมพ์แจกจ่าย
- CSV + QR Code เหมาะสำหรับใช้งานดิจิทัล
- ผู้ใช้เลือกได้ตามความต้องการ

### ฟีเจอร์ที่จะทำ:
1. **Export PDF** - พิมพ์แจกจ่าย
   - Layout เหมาะกับการพิมพ์
   - มี QR Code + ข้อมูล
2. **Export QR Codes (ZIP)** - ส่งผ่านช่องทางดิจิทัล
   - Export QR Code เป็นไฟล์ Image
   - Zip ไฟล์ทั้งหมด
   - แชร์ผ่าน LINE/Email
3. **Export CSV** - ใช้ในระบบอื่น
   - Export ข้อมูลพร้อม QR Code filenames

## 🚀 สิ่งที่จะทำต่อไป

1. ติดตั้ง library สำหรับสร้าง QR Code (qrcode.js)
2. สร้างฟังก์ชัน generate QR Code
3. สร้าง PDF template สำหรับ Export
4. เพิ่มปุ่ม "Export QR Codes" ในหน้า resident-accounts
5. เพิ่ม Dialog สำหรับเลือก Export options

## ความคิดเห็นอื่นๆ

**ทางเลือกอื่น**:
- ส่งผ่าน Email แบบอัตโนมัติ
- ส่งผ่าน SMS (ถ้ามี API สำหรับ SMS)
- สร้างหน้า Portal สำหรับ scan QR Code โดยตรง
